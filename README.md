# 龙门阵

龙门阵里话家常，  茶馆街头笑声扬。  
故事传奇传千古，  幽默讽刺韵味长。  
川渝文化显魅力，  民间智慧聚一堂。  
亲朋好友聚一堂，  龙门阵里乐陶陶。
